Twilio

Twilio is a cloud communications platform as a service (PaaS) company . Twilio allows software developers to programmatically make and receive phone calls and send and receive text messages using its web service APIs. Twilio's services are accessed over HTTP and are billed based on usage. If you want to use Messaging App with Backend Code you need to signUp on Twilio platform and to obtain accountSid ,authToken and one mobile Number credentials from Twilio.

Change the accountSid and authToken that you received for your account in the given path of the MessagingApp-Backend repository: /config/secretToken.js
